# # Password Generator

## Description

This is a simple Python Password Generator that creates strong and random passwords. The user enters the desired password length, and the program generates a password using letters, numbers, and special characters.

## Features

* Generates random passwords
* User can choose password length
* Uses letters, numbers, and symbols
* Easy to use

## Requirements

* Python 3.x

## How to Run

1. Save the code as `password_generator.py`.
2. Open a terminal or command prompt.
3. Run:

   ```
   python password_generator.py
   ```
4. Enter the desired password length.
5. The generated password will be displayed.

## Example

Input:

```
Enter password length: 12
```

Output:

```
Generated Password: A#7kP@2mX!9q
```
